# M7 P3 Replacement – Fruit of the Vine Latent Perfect Video Compilation

This repository contains the starter notebook and structure for our group project.

## Team Setup Instructions
The dataset is **not** included in this repository.  
Each team member must download the dataset from the shared Google Drive folder:

https://drive.google.com/drive/folders/1fPRwSA_L2gg7SLanIgUUFemcG14UfvMp

### Steps:
1. Create a local folder, e.g. `/home/<yourname>/vine_data/`
2. Download and unzip `embeddings.zip` into that folder
3. Ensure the file `vine_embeddings_plus.jsonl` is present
4. Update the `DATA_DIR` variable in `V5.ipynb` to match your local path
